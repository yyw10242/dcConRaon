## dccon_list Maker 사용법

1. DCConList_Maker\bin\Release 폴더의 DCConList_Maker.exe를 실행한다

2. 디시콘 이미지를 모아둔 폴더 경로를 적어준다.

3. 디시콘 이미지 폴더에 있는 dccon_list.js를 사용한다.
